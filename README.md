# .copilot

My personal skills, extensions, and configuration for [GitHub Copilot](https://github.com/features/copilot).

## Contents

### Skills

Custom skills that extend Copilot's capabilities:

- **address-pr-feedback** — Reviews and addresses pull request feedback
- **before-after-capture** — Captures before/after screenshots of iOS app changes
- **copilot-agents-api** — Queries the Copilot Agents API for task and session details
- **merge-main** — Merges latest changes from main and resolves conflicts
- **submit-pr** — Commits, pushes, and creates or updates pull requests
- **xcodebuildmcp** — iOS/macOS build, test, run, and debug via XcodeBuildMCP

### Extensions

- **copilot-agents-api** — Extension for interacting with the Copilot Agents API
